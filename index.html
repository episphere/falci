falci [code] [issues] [demo]

Modelling drug resistance in Plasmodium falciparum with interactive artificial intelligence (AI, using TensorFlow). The interactive modelling uses two methods, 1) deep learning AI and b) reference machine learning by a nearest neighbor classifier. The latter was implemented as part of the feasability study, and will be used as a reference to assess the AI prediction. If no trainign data is provided, a default data set will be used. The structure of this dataset also provides a template for user-provided data.
